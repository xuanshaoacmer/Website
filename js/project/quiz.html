    Game Rules

    1. Challenge any online player to a quiz duel.

    2. Both players answer the same question (multiple choice or true/false).

    3. Only your first submitted answer will be accepted.

    4. Correct answers earn 2 points.

    5. If both answer correctly, the faster player gets the points.

    6. If only one answers correctly, they get 2 points.

    7. The game ends when all questions are answered.

    8. The player with the higher total score wins.
